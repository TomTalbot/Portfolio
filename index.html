Projects

Terrain-demo

Terrain-demo is an adaptation of an algoritm for an unfinished pygame game. The algorithm makes use of the random library to draw terrain accordingly. It also uses seeds to allow a user to regenerate random terrain on the press of the spacebar.

BuildTool

The BuildTool is a terminal-based resource aiming to educate those with less knowledge of PC configuration and building. The tool comprises of a PC part picking feature, a complete guide on PC building, the ability to save parts to a file for future use, a way to view the most popular components and a way to view preconfigured builds.

Credit-card-validator

The Credit card validator is a simple tool to validate any given credit card number. The tool makes use of PyQT5 for the GUI and Luhn's algorithm to actually validate the number. The tool displays various techniques associated with PyQT.
